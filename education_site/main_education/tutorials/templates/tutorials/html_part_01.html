Heading & Paragraph

Heading in romana rubrica poate fi folosit pentru a scrie un text, cu o marime si un aspect diferit.

De exemplu, daca am nevoie de un text pentru un titlu de pagina putem folosii <h1></h1> tag

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title></title>
  </head>
  <body>
    <h1>Acesta este primul meu text!</h1>
  </body>
</html>

Bun, acum ca am scris "codul" in sectiunea body din fisierul HTML, sa il rulam si sa vedem cum arata el in browser

Bold Si Italic

Pentru acest stil de scriere sunt cunoscute doua metode pentru fiecare, voi prezenta metoda veche si noua.

Metoda veche de scriere pentru Strong si Italic

<p><i>Acest text este italic</i></p>

Liste

HTML are doua tipuri de liste:

  • Ordonate
  • Neordonate

    • Sa incepem cu lista ordonata

    Acest este definit prin OL tag.

    <ol>

</ol>

    OL (ordored list) reprezinta o lista numerotata.

    Pentru a creea un obiect(item) pentru o lista este necesar LI ce reprezinta un item dintr-o lista.

    <ol>
  <li>Primul obiect</li>
  <li>Al Doilea obiect</li>
</ol>
    • Title Page
    • file:///home/user/Fileroot/index.html

    2. Al Doilea obiect

    Se pare ca ele chiar sunt numerotate intre ele, dar daca nu avem nevoie de o lista numerotata, putem alege o lista neordonata.

    Pentru a folosi o lista neordonata avem nevoie de UL (unordered list) tag.

    <ul>
  <li>Text 1</li>
  <li>Text 2</li>
</ul>

    Poti combina UL tag cu UL tag

    <ol>
   <li>Text 1</li>
   <li>Text 2</li>
   <ul>
      <li>Lista neordonata in lista ordonata 1</li>
      <li>Lista neordonata in lista ordonata 2</li>
      <ul>
         <li>lel</li>
      </ul>
   </ul>
</ol>
